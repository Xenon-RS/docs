Xenon Racing Server [Version: W6R] 将进行维护 / 更新

## 维护时间

`2024/01/25 10:00 ~ 2024/01/25 15:00`  

## 维护内容

1. 更新服务器终端机部分，将支持**可自定义称号**的发放与领取  
2. 更新OCM活动 *第6次 第10届 副都心区域 涉谷-新宿*，更新完毕后同步开启活动
3. 数据库维护与备份

## 维护补偿

- 铭牌【夜桜】1副：将发放至您的终端机内，请于维护完成后的7天内领取
