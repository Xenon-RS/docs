          # 关于2023年......

          Writen by kKsk

          大约是在2022年还是2023年年初的时候（由于此前我正处于高三，因为进行高考，将湾岸放弃了大约2年的时间），我通过FST车队的一位好厚米了解到了湾岸存在私服。
          后来我也进去玩了（也就是QD服）。您或许会问：为什么我会选择自己开一个服务器？主要有以下原因：

          1. 专业需求 - 我大学录取的专业是软件工程，所以我很自然而然的就想发展自己技术能力。
          2. 游戏需求 - 因为QD的技术大佬实在太多，而我对湾岸这款游戏的玩法基本上就是属于休闲为主，所以就寻思能不能自己开服给自己随便玩玩怎样的。

          在偶然的一次看QD群的公告的时候，发现原来服务器是基于Bayshore的项目源码进行运营的。所以我就前往GitHub找到了项目，然后想着手研究。
          刚开始，我确实什么都不会，而且Bayshore的Wiki中对于部署服务器是讲述的非常模糊。于是请教了一下NicoChigu大佬。
          后来花了一些时间，研究明白了整体开服步骤。然后我就租用了一台服务器，后来也就开成了最初的服务器 --- OLG Server

          正常开服运营后，我和Chigu研究出了好些东西，也借用了低代码平台将排行榜站点与改车站开发完毕。
          或许是由于刚开始的改车站限制很宽，吸引来了一大批玩家前来游玩，当时没记错的话，最多的时候接近200人。

          但好巧不巧，玩家群体大了，遇到的困难也多了很多。为了保证玩家群体不要骂街或怎样的，我就弄了一些条例确保不要发生什么冲突。
          但是事与愿违，有些时候总会有意想不到的事情发生。

          大概在2023年的6月份左右（时间久远，我也不知道这个事情什么时候开始的），在我服务器内的三名玩家因代打问题起了冲突。
          事情是这样的：

          • 首先，我服务器的规定中并没有禁止代打，哪怕湾岸官方南梦宫都没有这种规定
          • 其次，事情主要原因还是在于UWU与Carie不知道找了谁给代打了TA成绩，然后莫名其妙KPGC就开始单向歧视
          • 由于莫名其妙的歧视，UWU与Carie又不是傻，所以就导致围绕了歧视而产生了多次冲突。

          这次事件也算是KPGC被踢出服务器是第一个问题所在。歧视没有问题，但正常来讲应该放在心里，偏要拿出来开怼，那就是影响了玩家环境。

          这次事件过后，大概在7月份左右，又发生了一次争吵。

          这次事情大概来龙去脉是这样的：

          UWU在给鬼王修正垫车化身，但是屡次遭到其他人的抢鬼
          后来UWU实在受不了，然后直接在群里开骂。后来我看消息的时候看到了，然后我就直接去服务器数据库查询到了对战的历史，发现UWU游玩期间，KPGC多次攻打同一区域导致改线
          为了维护玩家环境，UWU也一直在群里吵这件事，实在没办法，我只好将两人直接禁言加ban卡。

          后面我拉两个人到了一个群里，双方确认不能再有挑事行为发生，否则谁搞事我就ban谁。当时已经非常明白的声明过此事。
          同时，也清除了所有的排行榜记录，我希望能做到一笔勾销，避免再次有同样的事情发生。

          然而好景不长，在8月的时候，KPGC突然直接在群里挑事：说有人改他的车的ID，然后就又开始各种看不起。

          KPGC原本第一次发出来，我当作没有看到，就无所谓了。
          然后他又趁我在发言的时候再次发相同的信息出来，我实在忍无可忍，直接给他踢了。

          这里您或许好奇：为什么我会认定他这次是挑事？原因很简单。

          游戏的车辆的ID就如同你的身份证，整个中国跟你同名同姓的人多了去了，难不成因为有人撞名你就要去歧视他？
          其次，改名自由本身就是我们服务器的一个特点，你可以不爽，但是你必须接受。
          说白了一点，KPGC是希望利用撞名这一点，将他歧视的人给抹除。但必须得清楚：评判一个事情对不对，还得从根本说起。
          通过上述的分析，能确定的只有一点：KPGC利用这点进行挑事。

          所以结果很简单，我将他ban了，给踢掉了。

          然后就开始各种动荡：先是在我旅游的时候各种煽风点火搞到我只能急忙删群下线服务器以求那几天能平静一下，其次各种造谣，说我卷赞助款项跑路的也有，说我不负责任的也有，还给我服务器安个什么野鸡服的名号。

          讲真，我是不明白这样做有什么意义，反而看起来有种怨妇的感觉，得不到就毁灭的赶脚。

          讲个事情吧：我写这个文章的前阵子，手机圈玩刷机的圈子出了一单大事。本着我以前也玩刷机，我也去了解了一下事情。
          结果看到的是：LSPosed等著名开源玩机工具的GitHub仓库全部被Archived。
          原因很简单：某个游戏的开挂群体因为集体被封号然后就来骂只提供基本服务的LSPosed。

          说实在我也有感同身受，我运营这个服务器，钱是我出，技术维护也是我搞，玩家群体氛围也是我做，我似乎天天都在做好人。
          我开设了赞助，赞助这个东西，本身就是自愿，但是我愿意用游戏里的物品，去给玩家做一个赞助的回报。
          赞助总的来说不多，除去服务器费用，到手的款项也不多，当作辛苦费完全值得。

          可后面换来的是什么？我实实在在的给玩家服务，换来的是一片骂声，一片侮辱，还有大量玩家的流失。甚至乎，有些不知情的玩家，通过辱骂我们的人的口里，也认为我们服务器的本质就跟捞钱的没区别。
          我寻思，有必要吗？玩个游戏，搞那么多歧视，很值得吗？
          近几个月我一直寻求净化玩家群体，但搞着搞着玩家就剩下2、30个人了。明白我服务器整个事情的人少之又少，但是骂我们的人却很多。

          说实在的，我不希望服务器就这样就结束，后来给群里人劝了几次，我还是决定自己掏腰包也得继续开下去。

          这差不多就是OLG Server / Shimokitazawa / SeasonRC 的2023年。开始和中途都很好，但惨在被几个逆天给整成一锅烂粥。
          所以，我决定了一件事：2024年要重新做起。于是乎，根据原团队重新设定的Xenon就开始了。

          目前服务器还是基于Bayshore的分支 -- 由Kitsu编写的6R服务器的基础上修缮代码逻辑后运营。

          希望2024年能有个好的环境，不再出什么幺蛾子，我也不求那些对我们有误解的人能纠正误解。
          我只希望能将服务器运营好，提供给玩家稳定的服务，仅此而已。
          但还请不要乱搞歧视等行为，服务器的运营已经是属于用爱发电了，如果还要遭到各种辱骂，我不保证哪一天我可能真的会停掉服务器不了了之。但只要不影响我日常生活，我本人是没有什么所谓的。
